4. A QUÉ SE REFIERE EL ARCHIVO EN CONJUNTO
--------------------------------------------------------------------------------

El archivo contiene datos de telemetría del componente BootUX de Windows durante
el arranque o la entrada al Entorno de recuperación. Concretamente:

Contenido        Significado
---------        -----------
1920 × 1080      Resolución detectada para la interfaz de arranque (Full HD)
96 DPI           Escala estándar de Windows
ID 0x28DD        Eventos internos del menú de arranque (5 y 12: sin significado
                 documentado públicamente)

En resumen: registra cómo estaba configurada la pantalla cuando se mostró el
menú de recuperación, más algunos eventos internos del BootUX.

6. REFERENCIAS TÉCNICAS
--------------------------------------------------------------------------------

- BootUX: componente de Windows que dibuja la interfaz gráfica del arranque
  (menú de recuperación, opciones de inicio, etc.)

- bootux.dll y BootMenuUX.dll: DLLs que implementan esa experiencia de arranque

- IDs 0x28DD-0x28E0: identificadores de telemetría interna de Microsoft; no
  están documentados públicamente

- 1920×1080, 96 DPI: valores estándar que Windows usa para la interfaz de
  arranque cuando detecta una pantalla Full HD
