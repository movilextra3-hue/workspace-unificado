Desde aquí puedes ver y editar la información consolidada (CIS-TCOIN, Bank tokenization) y generar entradas para el registro. Los cambios se guardan en este navegador (localStorage).

  • Bank tokenization — Variables de entorno y checklist operativo para bUSDT/Vault.
  • Registro — Crear nueva entrada con formato estándar y copiarla al portapapeles para añadir a CONSOLIDACION_COMPLETA_TODO.md.

El registro completo está en CONSOLIDACION_COMPLETA_TODO.md (sección «Registro de interacciones»). Aquí solo se muestran entradas de ejemplo para búsqueda rápida.
